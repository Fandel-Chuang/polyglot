## 🔧 运行教程

### 运行方式：
```bash
# 编译并运行
./polyglot tutorial/tutorial1/hello.pg

# 或者进入目录运行
cd tutorial/tutorial1
../../polyglot hello.pg
```

## 📖 符号速查表

| 符号 | 含义 | 示例 |
|------|------|------|
| `>>` | 导入模块 | `>> "math"` |
| `?` | 变量声明 | `? name = "Alice"` |
| `@` | 结构定义 | `@ Point { x: f, y: f }` |
| `&` | 方法实现 | `& Point { ... }` |
| `%` | 接口定义 | `% Drawable { ... }` |
| `#` | 枚举/匹配 | `# Color { Red, Blue }` |
| `<-` | 函数返回 | `<- result` |
| `*` | 常量标记 | `* PI = 3.14` |
| `()? {}` | 条件语句 | `(x > 0) ? { ... }` |
| `^{}` | 循环语句 | `^(i < 10) { ... }` |
